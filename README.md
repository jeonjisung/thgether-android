### 설명

주변 공공 쓰레기통 위치를 확인할 수 있습니다. 또한, 공공 쓰레기통, 가로 휴지통이 등록되지 않은 지역에 직접 위치를 등록하여 사람들과 공유할 수 있습니다 !
